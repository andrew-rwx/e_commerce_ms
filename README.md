*E-shop di videogiochi usati*

Questo progetto è un e-shop pensato per la vendita e l'acquisto di videogiochi usati.
E' anche un primo esercizio nella programmazione di una applicazione che sfrutta i microservizi e un container nginx come reverse proxy.
